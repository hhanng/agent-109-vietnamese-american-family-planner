## 📋 Description
Family coordination tool built for Vietnamese-American households. Bilingual EN/VI throughout — chores tracker with point system, shared budget planner, family calendar with Vietnamese holidays, and a family hub with noticeboard and school phrase helper.

## ✨ Features
- **Chores** 🧹: 30 bilingual templates, assign to 6 family members, done toggle, weekly points leaderboard, chore wheel spinner
- **Budget** 💰: 12 bilingual categories, monthly expense tracking, progress bars (green/yellow/red), 10 Vietnamese-American money-saving tips
- **Calendar** 📅: Monthly view, Vietnamese + US holidays auto-added every year, event types color-coded, 7-day upcoming sidebar, doctor/school reminders
- **Family Hub** 👨‍👩‍👧: Sticky noticeboard, emergency contacts pre-filled (911, Poison Control, etc.), 20 school phrases EN/VI with 1-click copy, family goals with progress slider
- Fully bilingual — every label in English and Vietnamese 🇻🇳
- All data saved in localStorage

## 🛠 Tech Stack
- Vanilla HTML, CSS, JavaScript (single file, no dependencies)
- localStorage for all family data
